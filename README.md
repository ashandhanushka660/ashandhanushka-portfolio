# Ashan Dhanushka | Portfolio

Professional portfolio of **Ashan Dhanushka**, a Software Engineer and Creative Developer specializing in Fintech, Spring Boot, and Full-Stack Architectures.

## 🚀 Features
- **Dynamic Projects Showcase**: Visual and interactive project details.
- **Tech Stack**: Expertise in Java, Spring Boot, React, Next.js, and more.
- **3D Interaction**: Integrated Spline animations for a premium experience.
- **Responsive Design**: Optimized for all devices with Tailwind CSS.
- **Micro-animations**: Smooth Framer Motion transitions.

## 🛠️ Stack
- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [Spline](https://spline.design/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏁 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Deployment

The project is configured for **GitHub Pages**. The `docs/` folder is used for the static export.

To build the project:
```bash
npm run build
```

The output will be generated in the `docs/` folder, which can be served by GitHub Pages.
